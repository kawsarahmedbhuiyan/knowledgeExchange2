# KnowledgeExchange

## Overview
KnowledgeExchange is a web-based application that facilitates knowledge-sharing through forums, posts, and user interactions. It allows users to create forums, request enrollments, post discussions, and manage user authentication and authorization.

## Features
* User registration and authentication
* Forum creation, approval, and management
* Enrollment request handling
* Discussion posts and comments
* Role-based access control
* Logging and validation handling
